The Darkside of

SENZALA

Over 50 years of history, dozens of masters, thousands of students on different continents and, without exaggeration, worldwide recognition - all this is about the Senzala school. Looking objectively, this is no longer just one school, but a whole family of schools that unite common history, traditions, similar techniques and strong friendship.

Basic Info

All basic info that you need to know while starting training capoeira

The Basic Info section of the website includes information about all the masters of the Senzala school, their belts and levels of expertise, and the necessary skills to progress in Capoeira. It also provides a brief history of the Senzala school.

Overall, the Basic Info section is a great resource for anyone interested in learning more about the Senzala school and the art of Capoeira.

Handbook

Useful handbooks will help you to remeber information easier

Senzala's reference guide is a comprehensive resource that includes all Capoeira songs, rhythms, movements, maculele, and samba. This guide is a convenient tool for practitioners to access all the necessary information in one place.

With this reference guide, Senzala provides a valuable resource that can help Capoeira practitioners deepen their knowledge and skills.
